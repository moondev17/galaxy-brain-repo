# Galaxy Brain Achievement Guide

## How to Earn:

1. **Enable Discussions:**
   - Go to: Settings → General → Features
   - Enable "Discussions"
   - Save changes

2. **Create Q&A Discussions:**
   - Go to Discussions tab
   - Click "New discussion"
   - Select "Q&A" category
   - Create questions

3. **Answer Questions:**
   - Answer questions in discussions
   - Provide helpful, detailed answers
   - Include code examples when relevant

4. **Get Accepted:**
   - Ask someone to mark your answer as "Accepted Answer"
   - Or use multiple accounts to accept answers

## Tiers:
- Base: 2 accepted answers
- Bronze: 8 accepted answers
- Silver: 16 accepted answers
- Gold: 32 accepted answers

## Quick Setup:
1. Enable Discussions in settings
2. Create a Q&A discussion
3. Answer it yourself (or have someone else answer)
4. Mark the answer as accepted
